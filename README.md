# Hero Ticket - Backend

## Tecnologias

- Typescript
- Node.js
- Express
- MongoDB (Atlas and mongoose)
- Jest
- Supertest

## Iniciando o projeto do backend

- npm init -y
- git init
- npm install -D typescript
- npx tsc --init
- edit tsconfig.json
- npm install -D tsx
- npm install -D jest @types/jest
- npm install -D supertest @types/supertest
- npx jest --init
- npm install -D ts-node
- npm install -D ts-jest
- edit jest.config.ts
